# Clash Router macOS Release Assets

本目录保存 `Clash Router v2.5.1-router.1` 的 macOS 本地发布产物：

- `Clash Router.app`
- `Clash Router_2.5.1-router.1_aarch64.dmg`
- `Clash Router.app.tar.gz`

说明：

- `.app` 可直接本地运行
- `.dmg` 适合安装分发
- `.app.tar.gz` 是 updater 归档产物
- 若要启用 updater 正式签名，还需额外配置 `TAURI_SIGNING_PRIVATE_KEY`
